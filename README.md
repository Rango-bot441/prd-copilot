# PRD Copilot

PRD Copilot 是一个用于作品集展示的「需求澄清 Copilot + PRD Builder」小应用。它把一句模糊需求转化为需求成熟度评分、缺失信息、建议追问、需求卡和结构化 PRD 草稿。

## 核心功能

| 模块 | 输入 | 输出 |
| --- | --- | --- |
| 模糊需求输入 | 一句话需求、业务场景、需求阶段、补充信息 | 原始需求上下文 |
| 需求澄清 Copilot | 用户输入的需求信息 | 成熟度评分、已识别要素、缺失信息 |
| 追问生成 | 缺失的 PRD 要素 | 产品经理应继续追问的问题 |
| 需求卡 | 模糊需求 + 业务场景 | 用户、问题、目标、MVP、核心指标 |
| PRD Builder | 需求卡和澄清结果 | 完整 PRD 草稿 |

## 当前版本边界

- 当前为静态原型，不接入真实大模型。
- 成熟度评分基于关键词规则，不代表真实语义理解。
- PRD 输出是草稿，需要产品经理结合业务继续完善。
